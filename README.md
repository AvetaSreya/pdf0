# 📄 PDF0 – Privacy-Focused PDF Toolkit

A fast, privacy-focused PDF toolkit that runs entirely in your browser. No file uploads, no servers — your documents never leave your device.

---

## 🚀 Live Demo

👉 https://pdf0-alpha.vercel.app/

---

## 🧰 Tech Stack

* **Framework:** Next.js 16 (App Router)
* **UI:** React 19 + Tailwind CSS 4
* **Language:** TypeScript
* **PDF Processing:** pdf-lib + PDF.js

---

## ✨ Features

* 🔒 **100% Private** — All processing happens locally in your browser
* ⚡ **Fast Processing** — No upload/download delays
* 🆓 **Free to Use** — No login, no limits, no watermarks
* 📱 **Responsive Design** — Works on mobile and desktop

---

## 🛠️ Tools

| Tool           | Description                       |
| -------------- | --------------------------------- |
| Merge PDF      | Combine multiple PDFs into one    |
| Split PDF      | Extract specific pages from a PDF |
| Compress PDF   | Reduce file size                  |
| Rotate PDF     | Rotate pages (90°, 180°, 270°)    |
| PDF to Images  | Convert PDF pages to PNG or JPEG  |
| Images to PDF  | Create PDFs from images           |
| Organize Pages | Reorder, rotate, or delete pages  |
| Add Watermark  | Add text watermark to PDF pages   |

## ⚙️ Getting Started

### 📌 Prerequisites

* Node.js 18+
* npm / yarn / pnpm / bun

---

### 🔧 Installation

```bash
git clone https://github.com/AvetaSreya/pdf0.git
cd pdf0
npm install
npm run dev
```

👉 Open **http://localhost:3000**

---

## 🏗️ Build for Production

```bash
npm run build
```
